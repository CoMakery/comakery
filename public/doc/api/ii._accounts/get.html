II. Accounts API

Retrieve and manage account data, follow and unfollow projects.

GET

GET /api/v1/accounts/:id

Returns account data

Parameters

Name Description
id account id

Response Fields

Name Description
managed_account_id id
email email
firstName first name
lastName last name
nickname nickname
imageUrl image url
country country
dateOfBirth date of birth
ethereumWallet ethereum wallet
qtumWallet qtum Wallet
cardanoWallet cardano wallet
bitcoinWallet bitcoin wallet
eosWallet eos wallet
tezosWallet tezos wallet
createdAt account creation timestamp
updatedAt account update timestamp
verificationState result of latest AML/KYC verification (passed | failed | unknown)
verificationDate date of latest AML/KYC verification
verificationMaxInvestmentUsd max investment approved during latest AML/KYC verification
