II. Accounts API

Retrieve and manage account data, balances, interests.

CREATE VERIFICATION

POST /api/v1/accounts/:id/verifications

Returns account verifications (See VERIFICATIONS for response details)

Parameters

Name Description
id account id
verification[passed] verification result
verification[verification_type] verification type ( aml-kyc accreditation valid-identity ), defaults to aml-kyc
verification[max_investment_usd] maximum investment in us dollars
verification[created_at] timestamp

Response Fields

Name Description
errors array of errors
