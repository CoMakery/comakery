VI. Account Token Records API

Create and delete account token records, retrieve account token record data.

GET

GET /api/v1/projects/:project_id/account_token_records/:id

Returns data for a single account token record.

Parameters

Name Description
project_id project id
id account token record id

Response Fields

Name Description
id account token record id
account_id account id
token_id token id
reg_group_id reg group id
lockup_until lockup until
max_balance max balance
account_frozen account frozen
status account token record status (created synced)
createdAt creation timestamp
updatedAt update timestamp
