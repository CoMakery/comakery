VII. Blockchain Transactions API

Generate blockchain transactions for project to process and submit to blockchain.

GENERATE TRANSACTION

POST /api/v1/projects/:project_id/blockchain_transactions

Generates a new blockchain transaction for a transactable and locks the transactable for 10 minutes

Parameters

Name Description
project_id project id
transaction[source] transaction source wallet address
transaction[nonce] transaction nonce
blockchain_transactable_type transactable type to generate transaction for (awards, transfer_rules, account_token_records)
blockchain_transactable_id transactable id of transactable type to generate transaction for

Response Fields

Name Description
id transaction id
blockchain_transactable_id transactable id
amount transaction amount
destination transaction destination
source transaction source
nonce transaction nonce
contract_address token contract address
network token network
tx_hash transaction hash
tx_raw transaction in HEX
status transaction status (created pending cancelled succeed failed)
status_message transaction status message
createdAt transaction creation timestamp
updatedAt transaction update timestamp
