VII. Blockchain Transactions API

Generate blockchain transactions for project to process and submit to blockchain.

FAIL TRANSACTION

DELETE /api/v1/projects/:project_id/blockchain_transactions/:id

Marks transaction as failed and excludes transfer from further transactions, see GENERATE TRANSACTION for response fields

Parameters

Name Description
project_id project id
id transaction id
transaction[tx_hash] transaction hash
transaction[status_message] transaction status message
transaction[failed] marks transaction as failed, to exclude the transactable from further transactions
