IX. Wallets API

Create, update, delete and retrieve account wallets.

UPDATE WALLET

PUT /api/v1/accounts/:id/wallets/:wallet_id

Returns updated wallet (See INDEX for response details)

Parameters

Name Description
id account id
wallet_id wallet id
primary_wallet primary wallet flag
