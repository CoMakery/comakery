Parameters

Name Description
address wallet address
blockchain wallet blockchain ["bitcoin", "bitcoin_test", "cardano", "cardano_test", "qtum", "qtum_test", "eos", "eos_test", "tezos", "constellation", "constellation_test", "ethereum", "ethereum_ropsten", "ethereum_kovan", "ethereum_rinkeby", "algorand", "algorand_test", "algorand_beta"]
source wallet source ["user_provided", "ore_id"]
tokens_to_provision array of params to tokens provision
tokens_to_provision["token_id"] token_id is required if tokens_to_provision provided
tokens_to_provision["max_balance"] max_balance is required for security tokens only

tokens_to_provision["account_frozen"] account_frozen is required for security tokens only
