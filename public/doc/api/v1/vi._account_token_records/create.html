VI. Account Token Records API

Create and delete account token records, retrieve account token record data.

CREATE

POST /api/v1/projects/:project_id/account_token_records

Returns created account token record details (See GET for response details)

Parameters

Name Description
project_id project id
account_token_record[max_balance] max balance
account_token_record[lockup_until] lockup until
account_token_record[reg_group_id] reg group id
account_token_record[account_frozen] frozen

Response Fields

Name Description
errors array of errors
