XII. Hot Wallet Addresses API

Register hot wallets with a project.

CREATE HOT WALLET

POST /api/v1/projects/:project_id/hot_wallet_addresses

Returns created hot wallet

Parameters

Name Description
name wallet name ("Hot Wallet" by default)
address wallet address

Request

Headers

Api-Transaction-Key: 28ieQrVqi5ZQXd77y+pgiuJGLsFfwkWO

Route

POST /api/v1/projects/11111111/hot_wallet_addresses

}
