IV. Transfers API

Create and cancel transfers, retrieve transfer data.

GET

GET /api/v1/projects/:project_id/transfers/:id

Returns data for a single transfer.

Parameters

Name Description
project_id project id
id transfer id

Response Fields

Name Description
id transfer id
amount transfer amount
quantity transfer quantity
totalAmount transfer total amount
description transfer description
accountId transfer account id
transferTypeId transfer type id
transactionError latest recieved transaction error (returned from DApp on unsuccessful transaction)
status transfer status (accepted paid cancelled)
recipientWalletId recipient wallet id
createdAt transfer creation timestamp
updatedAt transfer update timestamp
