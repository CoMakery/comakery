V. Transfer Rules API

Create and delete transfer rules, retrieve transfer rules data.

CREATE

POST /api/v1/projects/:project_id/transfer_rules

Returns created transfer rule details (See GET for response details)

transfer_rule[receiving_group_id] receiving reg group id
transfer_rule[lockup_until] lockup until
