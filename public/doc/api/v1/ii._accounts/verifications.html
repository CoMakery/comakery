II. Accounts API

Retrieve and manage account data, balances, interests.

VERIFICATIONS

GET /api/v1/accounts/:id/verifications

Returns an array of verifications

Parameters

Name Description
id account id
page page number

Response Fields

Name Description
passed verification result
verification_type verification type ( aml-kyc accreditation valid-identity ), defaults to aml-kyc
maxInvestmentUsd maximum investment in us dollars
createdAt timestamp
