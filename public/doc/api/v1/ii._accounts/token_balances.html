II. Accounts API

Retrieve and manage account data, balances, interests.

TOKEN BALANCES

GET /api/v1/accounts/:id/token_balances

Returns an array of token balances

Parameters

Name Description
id account id

Response Fields

Name Description
total_received total received on platform
total_received_and_accepted_in total received and accepted on platform
blockchain[address] blockchain address
blockchain[balance] blockchain balance
blockchain[maxBalance] blockchain max balance
blockchain[lockupUntil] blockchain lockup until
blockchain[accountFrozen] blockchain account frozen status
token[id] token id
token[name] token name
token[symbol] token symbol
token[network] token network ( main | ropsten | kovan | rinkeby )
token[contractAddress] token contract address
token[decimalPlaces] token decimal places
token[logoUrl] token logo url
token[createdAt] token creation timestamp
token[updatedAt] token update timestamp
